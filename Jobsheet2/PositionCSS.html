position : static

Position static digunakan untuk mengatur elemen menjadi statis secara default. elemen akan mengikuti posisi normal secara default, elemen tidak dipengaruhi oleh properti top, bottom, left, dan right

elemen ini diatur menggunakan posisi static;




position : relative

sebuah elemen HTML yang menggunakan position reatie akan terletak pada posisi normal. mengatur properti atas, anan, bawah, dan kiri dari elemen yang di posisikan dengan "relatif" akanmebuat jauh dari posisi normalnya. konten lain tidak akan di sesuaikan agar sesuai dengan celah yang ditinggalkan oleh elemen tersebut

elemen ini diatur menggunakan posisi : relative



position : fixed

sebuah elemen HTML yang di setting dnegan position fixed akan memiliki sifat tetap, tanpa ada perubahan bahkan jika halaman website di scrool. berlaku pengauran left, bottom, top, dan right pada position fixed.

elemen ini diatur menggunakan posisi : fixed















position : absolute

elemen HTMMl yang menggunakan position absolute akan diposisikan relatif dengan elemen lain yang mendahuluinya yang terdekat, bukan relatif terhadap layar secara normal.

elemen ini diatur menggunakan posisi : relative
elemen ini diatur menggunakan posisi : abosolute






position : sticky

cobalah untuk scroll untuk mengetahui bagaimana posisi sticky bekerja.

catatan : IE/Edge 15 dan versi yang lebih lama tidak mendukung sticky

hola aku adalah sticky!
